# Lodistream Legal - Cloudstream Yasal Eklenti Deposu

Bu depo yalnızca **yasal, kamuya açık veya Creative Commons** lisanslı içerikler sağlar.

### 🧩 Eklentiler:
- 🎬 **Trailers** – IMDb/TMDB/YouTube fragmanları  
- 🎥 **Documentaries** – Creative Commons belgeseller  
- 🎞️ **Archive Movies** – Public Domain filmler  

### 📦 Cloudstream’e ekleme:
1. Cloudstream aç  
2. Ayarlar → Eklentiler → Depo ekle  
3. Şu URL’yi yapıştır:  
   ```
   https://raw.githubusercontent.com/lodistream/lodistream-legal/main/plugins.json
   ```
4. Kaydet ve eklentileri yükle 🎉
